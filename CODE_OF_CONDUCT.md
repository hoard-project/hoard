# Code of Conduct

Hoard follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported by contacting the project maintainers at
[hoard-project.team@zvip.eu.org](mailto:hoard-project.team@zvip.eu.org).
